# RWA Lending Platform - Frontend

React application for RWA Lending Platform.

## Setup

1. Copy contract addresses from `backend/deployments/localhost.json` to `src/config/contracts.js`
2. Copy ABIs from `backend/artifacts/contracts/` to `src/abis/`

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
